## Install azurefile CSI driver v1.8.1 version on a Kubernetes cluster

### Install by kubectl
```console
curl -skSL https://raw.githubusercontent.com/kubernetes-sigs/azurefile-csi-driver/v1.8.1/deploy/install-driver.sh | bash -s v1.8.1 --
```

 - check pods status:
```console
kubectl -n kube-system get pod -o wide --watch -l app=csi-azurefile-controller
kubectl -n kube-system get pod -o wide --watch -l app=csi-azurefile-node
```

example output:


### clean up Azure File CSI driver
```console
curl -skSL https://raw.githubusercontent.com/kubernetes-sigs/azurefile-csi-driver/v1.8.1/deploy/uninstall-driver.sh | bash -s --
```
